# Ảnh Mít (mood + item)

Thả các file PNG với **đúng tên** dưới đây vào `src/assets/hamsters/` **hoặc**
`src/assets/mit/` — code quét cả hai thư mục qua `import.meta.glob`, không cần sửa
gì thêm. Khi thiếu file nào, app tạm dùng ảnh hamster cũ (mood/phản ứng) hoặc
emoji (món đồ kéo-thả) làm fallback.
